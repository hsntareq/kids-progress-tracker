# Product Requirements Document (MVP)

## Product Name
Family Quest MVP

## Vision
Enable families to create positive routines where children complete educational or activity tasks, earn points, and spend those points through parent-approved requests.

## Primary Personas
- Parent: creates milestones/tasks, approves spending requests.
- Child: updates profile, completes tasks, tracks performance.
- Admin: accesses parent capabilities and future tenant oversight.

## Core User Stories
1. As a parent, I can sign in and assign tasks with point values to my child.
2. As a child, I can view assigned tasks and mark them complete.
3. As a child, I can see my performance out of 100 in a chart.
4. As a child, I can submit a spending request for points.
5. As a parent, I can approve or deny spending requests.
6. As a parent, I can create milestones and monitor progress.

## MVP Scope
- Authentication: Google + email/password.
- Role-based profile data.
- Points accrual from completed tasks.
- Spending request workflow.
- Milestone tracking using point progress.
- Responsive dashboard and profile pages.

## Non-Goals (MVP)
- Subscription billing.
- Push notifications.
- External integrations (calendar/LMS).
- Native app implementation.

## Success Metrics
- Time-to-first-task created < 5 minutes.
- 80% of tasks are completed within assigned periods.
- Parent response SLA to requests < 24 hours.
